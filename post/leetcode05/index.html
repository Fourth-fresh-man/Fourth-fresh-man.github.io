Leetcode刷题 - 05 替换空格

2020-12-12

实现一个函数，把字符串s中的每个空格替换成“%20”

示例：

输入：s = "We are happy."
输出："We%20are%20happy."

思路

方法一 调库

class Solution {
    public String replaceSpace(String s) {
        return s.replaceAll(" ","%20");
    }
}

方法二 自定义新字符串

class Solution {
    public String replaceSpace(String s) {
        StringBuilder sb = new StringBuilder();
        // 扫描字符串
        for (int i = 0; i<s.length();i++){
            // 遇到空格，则添加%20
            if (s.charAt(i) ==' '){
                sb.append("%20");
            }else {
            // 不是空格直接添加
                sb.append(s.charAt(i));
            }
        }
        return sb.toString();
    }
}
