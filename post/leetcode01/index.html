Leetcode刷题-01 左旋转字符串

2020-12-09 字符串 力扣 简单

字符串的左旋转操作是把字符串前面的若干个字符转移到字符串的尾部。请定义一个函数实现字符串左旋转操作的功能。比如，输入字符串"abcdefg"和数字2，该函数将返回左旋转两位得到的结果"cdefgab"。

示例 1：

输入: s = "abcdefg", k = 2
输出: "cdefgab"

示例 2：

输入: s = "lrloseumgh", k = 6
输出: "umghlrlose"

限制：
1 <= k < s.length <= 10000

思路

方法一 字符串切片

利用Java的字符串API来实现

public static String round_str1(String str, int num){
        // 分成0-n和n-str.length两个长度字符串，并拼接
        return str.substring(num, str.length())+str.substring(0,num);
}

方法二 列表遍历

利用列表获取字符串指定位置元素

public static String round_str2(String str, int num){
    // 利用空字符串构造器，将str指定位置上的字符逐个传入
    StringBuilder res = new StringBuilder();
    for (int i = num; i < str.length(); i++) {
        res.append(str.charAt(i));
    }
    for (int i = 0; i < num; i++) {
        res.append(str.charAt(i));
    }
    // 返回新字符串
    return res.toString();
}

墙裂推荐！求余计算

简化代码

public static String round_str2(String str, int num){
    StringBuilder res = new StringBuilder();
    for (int i = num; i < num + str.length(); i++){
        res.append(str.charAt(i%num));
    }
    return res.toString();
}

方法三 字符串遍历拼接

public static String round_str(String str, int num){
    String res = "";
    for (int i = num; i < num + str.length(); i++) {
        res+=str.charAt(i % str.length());
    }
    return res;
}
